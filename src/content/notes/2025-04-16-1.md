Stop vertically aligning your checkboxes with `center`. Insteaqd use `baseline` instead to keep it aligned with the first line of the label text.

```scss
label {
  display: flex;
  align-items: center; // [!code --]
  align-items: baseline; // [!code ++]
}
```
