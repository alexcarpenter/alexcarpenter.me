---
title: Mouse Tracking with CSS Variables
description: Use CSS Variables powered by a few lines of JavaScript to have elements follow the cursor.
publishedAt: '2021-01-12'
tags: [JavaScript]
---

<ExternalLink href="https://codepen.io/team/keyframers/pen/qBZdeeq" />

```js
const app = document.querySelector('#app');

document.body.addEventListener('mousemove', (event) => {
  app.style.setProperty('--x', event.clientX);
  app.style.setProperty('--y', event.clientY);
});
```
